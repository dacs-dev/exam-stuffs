# Exam-Stuffs

A collection of solutions and practice exercises for the **42 School** exam ranks. This repository contains personal implementations of common exam problems, used to prepare for and pass the 42 curriculum exams.

## Contents

| Folder | Description |
|--------|-------------|
| `exam_rank_2/` | Solutions for Exam Rank 02 — C programming fundamentals |
| `exam_rank_4/` | Solutions for Exam Rank 04 — Advanced C exercises |
| `exam_rank_5/` | Solutions for Exam Rank 05 — C++ exercises |

## About 42 Exams

The 42 School curriculum includes periodic exams at various ranks. Each exam requires completing programming exercises within a set time limit, without access to the internet. These exams test core skills including:

- Memory management and pointers (C)
- Data structures and algorithms
- Object-oriented programming (C++)
- Unix system calls and processes

## Notes

These solutions are intended for personal reference and learning. If you are currently studying at 42, try to solve the exercises yourself first — that is the best way to actually learn.
